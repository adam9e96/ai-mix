# AI-MIX

AI 기반 학습 플랫폼

## 📖 프로젝트 소개

AI-MIX는 사용자들이 AI와 대화하고, 질문과 답변을 공유하며, 배틀 모드를 통해 학습할 수 있는 AI 기반 학습 플랫폼입니다. GPT를 활용한 챗봇 기능과 커뮤니티 기반의 Q&A, 게임화된 학습 경험을 제공합니다.

## ✨ 주요 기능

- **AI 챗봇**: GPT 기반 대화 기능으로 실시간 질문과 답변
- **Q&A 게시판**: 사용자 간 질문과 답변 공유 커뮤니티
- **배틀 모드**: 문제 풀이를 통한 게임화된 학습 경험
- **지식 카드**: 위키 형태의 지식 저장소 및 관리

## 🛠️ 기술 스택

### Frontend
- **React 19** - 사용자 인터페이스 구축
- **Vite** - 빌드 도구 및 개발 서버
- **React Router DOM** - 클라이언트 사이드 라우팅
- **Zustand** - 상태 관리
- **Framer Motion** - 애니메이션
- **React Markdown** - 마크다운 렌더링
- **React Flow** - 노드 기반 시각화
- **Recharts** - 데이터 시각화
- **Axios** - HTTP 클라이언트
- **React Toastify** - 알림 메시지

### Backend
- **Spring Boot 3.5.9** - REST API 서버
- **Java 21** - 백엔드 개발 언어
- **Spring Data JPA** - 데이터베이스 ORM
- **Spring Security** - 인증 및 보안
- **JWT** - 토큰 기반 인증
- **PostgreSQL** - 관계형 데이터베이스
- **Redis** - 캐싱 및 세션 관리
- **OpenAI Java SDK** - GPT API 연동
- **MapStruct** - DTO 매핑
- **Lombok** - 보일러플레이트 코드 감소
- **Gradle** - 빌드 도구

### Infrastructure
- **PostgreSQL** - 메인 데이터베이스
- **Redis** - 캐싱 레이어
